# 🚀 Aras'ın İlk Web Sitesi

Bu benim web geliştirme dünyasına attığım **ilk adım!** 🖐️ Profesyonel bir çalışma olmasa da, HTML, CSS ve JavaScript temellerini öğrenmek için geliştirdiğim, benim için çok değerli bir proje.

## ✨ Özellikler

Basit ama içinde eğlenceli teknolojiler barındırıyor:

*   **Ziyaretçi Defteri:** Discord Webhook kullanarak mesajlarınızı doğrudan Discord kanalıma iletiyor.
*   **Spotify Entegrasyonu:** Şu an ne dinlediğimi canlı olarak gösteriyor (Spotify API).
*   **Tema Desteği:** İster karanlık ister aydınlık modda gezinebilirsin. 🌙/☀️
*   **Sekmeli Yapı:** Hakkımda, Günlük ve Oyunlar gibi farklı bölümler arasında hızlıca geçiş yapabiliyorsun.

## 🛠️ Kullanılan Teknolojiler

*   **HTML5:** Sayfa yapısı ve içerik.
*   **CSS3:** Tasarım, renkler ve basit animasyonlar.
*   **JavaScript (Vanilla):** Dinamik içerik yükleme ve API etkileşimleri.

## ⚙️ Kurulum ve Çalıştırma

Bu projeyi kendi bilgisayarınızda çalıştırmak için:

1.  Dosyaları bilgisayarınıza indirin.
2.  `index.html` dosyasını herhangi bir tarayıcıda açın.
3.  (Opsiyonel) Spotify ve Discord özelliklerinin çalışması için `.env.example` dosyasındaki alanları doldurup API anahtarlarınızı eklemeniz gerekir.

## 📝 Not

Bu site benim öğrenme sürecimin bir parçası. Kodlar belki mükemmel değil ama her satırında bir şeyler öğrenmenin heyecanı var. Bakalım bir sonraki versiyon nasıl olacak! ⚡
